# Portfolio

Case studies from real projects I built or worked on while transitioning from
automation/robotics engineering into cybersecurity (targeting Junior SOC
Analyst / Junior OT-ICS Security Analyst roles).

Source code for these projects lives in private/proprietary repositories and
is intentionally not included here. Each write-up covers the task, my role
and approach, the tech stack, and the results — no source code.

## Case studies

- [E-commerce catalog automation (Shoper / SUPON Kielce)](shoper-supon-catalog-automation.md) — Node.js scripts against the Shoper REST API to restructure, import, and convert a B2B store's product catalog at scale.
- [Internal B2B client portal (SUPON Kielce)](supon-b2b-portal.md) — Next.js platform with admin/client panels, role-based access, order and ticket lifecycles, and reporting.
- [Workwear/laundry tracking application](laundry-tracking-app.md) — Python/Tkinter desktop app with RFID hardware integration for tracking workwear issuance across companies and departments.
